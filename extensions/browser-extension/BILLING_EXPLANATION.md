# ChatGPT Plus vs API Billing - Important Explanation

## ⚠️ Critical Understanding

**Your ChatGPT Plus subscription ($20/month) does NOT cover API usage.**

These are **two separate billing systems**:

### 1. ChatGPT Plus Subscription ($20/month)
- ✅ Access to ChatGPT web UI (chatgpt.com)
- ✅ Priority access during peak times
- ✅ Access to GPT-4 model in web UI
- ❌ **Does NOT include API usage**

### 2. OpenAI API Usage (Pay-as-you-go)
- ✅ Separate billing based on tokens used
- ✅ Typically $0.01-0.03 per 1,000 tokens
- ✅ Much cheaper than subscription for most users
- ✅ Requires separate payment method setup

## How Billing Works

### Scenario: User with ChatGPT Plus

**User has:**
- ✅ ChatGPT Plus subscription ($20/month) → Covers web UI only
- ❌ **No API access** → Need to set up separately

**To use Spectyra:**

1. Go to [platform.openai.com](https://platform.openai.com)
2. Create account
3. Add payment method
4. Create API key
5. **Only pay for API usage** (no subscription needed)

**Example Costs:**

## Setting Up API Access

### Step 1: Create API Account
1. Go to [platform.openai.com](https://platform.openai.com)
2. Sign in (or create account)
3. **Note**: This is separate from your ChatGPT Plus account (though you can use same email)

### Step 2: Add Payment Method
1. Go to "Billing" → "Payment methods"
2. Add credit card
3. **Important**: This is separate from ChatGPT Plus billing

### Step 3: Set Spending Limits (Recommended)
1. Go to "Billing" → "Usage limits"
2. Set monthly limit (e.g., $50/month)
3. This prevents unexpected charges

### Step 4: Create API Key
1. Go to "API keys"
2. Click "Create new secret key"
3. Copy key immediately
4. Paste into Spectyra settings
