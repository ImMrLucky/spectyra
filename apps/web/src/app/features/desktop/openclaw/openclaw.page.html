OpenClaw + Spectyra

OpenClaw is a local CLI agent that supports custom provider endpoints. Spectyra's Local Companion acts as an in-path optimization proxy — OpenClaw sends requests to Spectyra, which optimizes, then calls your real provider.

How it works
OpenClaw → Spectyra (localhost:4111) → Your provider (OpenAI / Anthropic)

Inference goes direct to your provider. No Spectyra cloud relay. Billing stays on your account.

Troubleshooting

Common issues and fixes.

Companion not reachable

Make sure the Spectyra desktop app is open. The companion auto-starts on launch. Check that port 4111 isn't in use by another process.

OpenClaw not using Spectyra

Verify your config file has the custom provider pointing to http://localhost:4111/v1. Restart OpenClaw after config changes.

API key errors

The companion needs your provider API key. Go to Settings to configure it.
