Install OpenClaw

Get OpenClaw running on your machine first.

Where to run: Open Terminal (Applications → Utilities → Terminal), or a terminal panel in your editor. Paste the commands below into that window — or use Run OpenClaw onboard to open Terminal and run them for you.

Where to run: Open PowerShell (Windows — official OpenClaw installer uses PowerShell). Paste the commands below — or use Run OpenClaw onboard to run them in a new PowerShell window. Git Bash is optional if you prefer a Unix-style shell for manual steps.

Where to run: On macOS/Linux use Terminal with the bash line below; on Windows use PowerShell with the PowerShell line. Or use WSL on Windows if you prefer bash.

Troubleshooting

Common issues and fixes.

Companion not reachable

Make sure the Spectyra desktop app is open. The companion auto-starts on launch. Check that port 4111 isn't in use by another process.

OpenClaw not using Spectyra

Verify your config file has the custom provider pointing to http://localhost:4111/v1. Restart OpenClaw after config changes.

API key errors

The companion needs your provider API key. Go to Desktop settings to configure it.
