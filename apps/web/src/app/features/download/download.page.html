Installers not configured yet

Desktop download URLs are not set on the API yet. Your team can add them by setting DESKTOP_DOWNLOAD_MAC_URL and DESKTOP_DOWNLOAD_WINDOWS_URL on the Spectyra API server (e.g. Railway), pointing at your hosted .dmg and Windows installer.

Until then, use the Local Companion (terminal) with the same account — see Integrations for setup.

Other ways to run Spectyra

  • Integrations — SDK, browser extension, OpenClaw-style base URL, and companion.
  • Local Companion: npx @spectyra/local-companion then set OPENAI_BASE_URL=http://127.0.0.1:4111/v1.
