Spectyra for OpenClaw

Local companion, live token savings, spectyra/* models

Run Spectyra next to OpenClaw: optimize LLM traffic through a small process on your machine, then watch estimated savings in real time on the local dashboard. Install the skill from ClawHub, then install and start the Local Companion.

Install and start the Local Companion

npm install -g @spectyra/local-companion@latest && spectyra-companion start --open

After it starts, open the real-time savings dashboard: http://127.0.0.1:4111/dashboard

Real-time savings

The Local Companion serves a dashboard on 127.0.0.1:4111 so you can see token and cost savings as you use OpenClaw—no separate billing flow required for local use.

Local-first

Model traffic is optimized on your machine; your LLM provider credentials stay in your Local Companion configuration—not sent to Spectyra for inference.
