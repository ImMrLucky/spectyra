Session analytics

What this page is: rollups from redacted session summaries synced to Spectyra when a client turns on cloud sync (tokens, estimated costs, transforms — not raw prompts). Data is grouped by product lane so OpenClaw / Local Companion is not mixed with in-app SDK session sync.

Not the same as “Usage” in the sidebar: Workspace usage is your plan, limits, and billing snapshot. Per-call SDK telemetry (API key, per environment) lives under Projects → open a project — that path does not use this session sync table.

When does sync happen? Only when you enable it in a client that supports it — for example Local Companion’s sync-to-cloud setting, or the Spectyra desktop app posting session summaries while signed in. If you never turned that on, zeros here are expected.

In-app SDK without desktop: you do not need the desktop app. Use a Spectyra API key with POST /v1/telemetry/run (or the SDK’s cloud telemetry options) and view rollups on Projects. Session analytics here only appears if something also syncs session payloads.
