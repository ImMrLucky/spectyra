Security & Data Handling

Inference Path

Direct to provider. Spectyra never proxies your prompts or responses through Spectyra servers.

Provider Billing

Your provider account. Spectyra does not pay for or intercept provider usage.

Telemetry

Local only by default. Aggregated analytics can optionally be synced to Spectyra cloud.

Prompt Storage

Before/after prompt snapshots are stored locally. They are never uploaded unless you explicitly opt in.

Cloud Relay

None. No live inference traffic is routed through Spectyra servers.

OpenClaw Setup

No code integration required

Spectyra Local Companion runs beside OpenClaw on your machine. Point OpenClaw to the local companion endpoint.

  1. Start the Spectyra Local Companion (it's running now if you see the green dot)
  2. In your OpenClaw configuration, set the API base URL to:
    http://127.0.0.1:4111/v1
  3. Set your provider API key in OpenClaw as normal (it stays on your machine)
  4. Choose run mode: observe (recommended to start) or on
  5. Run a task and check the Savings page for results

What stays local

  • All prompts and responses
  • Your provider API key
  • Before/after prompt comparisons
  • Run analytics
